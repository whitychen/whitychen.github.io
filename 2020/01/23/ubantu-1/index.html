Ubuntu 18.04.3 LTS + Win10 双系统安装全过程

2020年1月 张为

本篇无图，所以不适用于初学的教程，但是如果遇到一些问题，可以根据这里面的步骤、相关名词等信息查阅网上资料。

Win10

没啥好说的。一般电脑都会预装正版，不用删。如果你的电脑里什么系统都没有的话，先装Windows。这种闭源的东西要安装我一般也都交给一键完成的软件做了，找一个绿色点的系统、绿色点的USB启动盘制作软件就行。

磁盘分区

Win10安装好之后可以对磁盘分一下区。右键点击左下角的Windows标志，选择磁盘管理器，进行卷的压缩、增添、删除，建议把给Ubuntu留的地方都不要设置分区，一块硬盘上Windows本身的分区最好相邻。在Windows的安装盘（efi启动分区和C盘所在硬盘）至少要留出200MB的空地方，留给Ubuntu的启动引导用。

Ubuntu的USB安装盘制作

清华tuna镜像站里能下载到Ubuntu发行版安装包，下载速度可能比官网快一点。选择对应版本的desktop版（amd64是正确的），得到一个iso镜像文件。版本上LTS版本会有更长时间的支持，然后如果太新推出来的版本也不要选，可能获得的社区支持太少，有问题不知道怎么解决。

Ubuntu 18.04.3 LTS的安装

打开电脑，试图打开BIOS（Esc, Del, F1, F2, F10, F12都可能是BIOS启动按键）。如果你发现不闪主板厂商/整机厂商的标志直接进Windows的话，可能需要再Windows里关闭快速启动/安全启动。

假设你已经成功进入了BIOS，在启动顺序里，把USB启动放到最前面，或者也可能是外置CD/DVD启动，也放到前面，重启一下。如果有问题就尝试把安全启动 (Secure boot) 关掉。

正确的情况应当是得到试用Ubuntu、安装Ubuntu和一些其它选项。担心出问题的可以先试用一下，试用完毕后双击桌面上的安装程序开始安装。安装的时候先选择语言地区网络，然后到了安装选项，一般就选正常安装 (normal) 就行，如果希望对软件有更大的掌控的话选最简安装 (minimal)，底下的两个勾选（是否下载更新和是否安装一些第三方软件）可选可不选。我在安装的时候选择的语言是英语，地区点在了中国，貌似官方源设置在国内了 (cn.archive.ubuntu.com)，下载速度应该不会太慢。

关键的是安装类型。如果已经有其它版本或者同版本的Ubuntu打算覆盖或者重装的话，一般不需要改变分区和挂载点，点擦除之前安装过的Ubuntu并重装就行了，很省心，如果没有已经安装的Ubuntu的话，如果打算安装在与Windows系统盘相同的硬盘上，并且信任自动分区的话，可以选择和Windows装在一起的选项，如果不信任或者打算装在其它硬盘上的话，建议是选择最下面的其它 (something else)，进入手动选择分区。手动需选择的时候空的 (free space)是没有分区过的，其它的是已经分区过的，如果选已经分区过的要小心其中的数据会丢失。有下面这么几个目录需要选择挂载点。

目录 建议大小 格式 描述
/ 20—30G ext4 根目录，全局和系统文件所在位置。大小不要太小就行，如果磁盘空间大可以分配更多。
/boot 200M ext4 启动引导程序。必须和Windows启动引导程序放在同一块硬盘下。
swap 8G swap 交换空间。
/tmp 5G ext4 系统临时文件。
/home 尽量大 ext4 所有用户的个人目录。如果没有特殊情况的话大部分数据都在这里了。

双系统启动引导

在BIOS里把两个系统启动引导分区对应硬盘的启动顺序放在最前面，如果可以选的话Ubuntu应该在Windows前面，GRUB引导启动。正常情况下应该出现GNU GRUB的界面，第一个选项是Ubuntu，后面会有Windows的选项。如果这样启动不起来的话把Windows放回前面，在Windows系统下尝试EasyBCD等工具修复启动引导。

至此，系统已经安装完毕可以使用，接下来一篇我会写一些我比较喜欢的配置，包含的会比较杂，有些也是积累了很久的配置，希望能有一两个配置有所帮助，我就很满意了。
